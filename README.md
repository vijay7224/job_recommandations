💼 AI-Based Job Recommendation System (Flask)

An AI-powered web application that recommends suitable job roles to users based on their skills or resume using Machine Learning and NLP techniques, built with a Flask web interface.

🚀 Features
🧠 AI-based job recommendations
📄 Resume or skill-based input
🔍 Skill extraction using NLP
🎯 Personalized job suggestions
🌐 Web interface using Flask
⚡ Fast and efficient predictions

🛠️ Tech Stack
Frontend: HTML, CSS
Backend: Flask (Python)
Machine Learning: scikit-learn
Libraries: pandas, numpy, nltk, pickle

⚙️ How It Works
User enters skills or uploads resume
System extracts important keywords
Convert text into vectors (TF-IDF / CountVectorizer)
Compute similarity with job dataset
Display top recommended jobs

🧠 Algorithm Used
Content-Based Filtering
Cosine Similarity
TF-IDF Vectorization
